# NIKE Air Design 

A premium UI/UX design exploration for **Nike Air**, meticulously crafted in **Figma**. This project focuses on a sleek, modern, and high-performance aesthetic that aligns with the Nike brand identity.

##  Design Preview

![Nike Air Design Preview](imgs/design.png)

## Key Features

- **Modern UI Architecture**: Clean lines and dynamic layouts designed for a premium user experience.
- **Brand-Centric Aesthetic**: Deep integration of Nike's iconic visual elements and typography.
- **Interactive Elements**: Thoughtful transitions and component states explored in the design.
- **High-Fidelity Assets**: Utilization of high-definition imagery and custom iconography.

##  Tools Used

- **Figma**: Primary design and prototyping tool.
- **Adobe Photoshop**: Image processing and retouching.
